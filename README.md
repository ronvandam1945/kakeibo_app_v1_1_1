# 家計簿管理アプリ（Spring Boot）



## 概要
Spring Boot を用いて作成した、ログイン機能付きの家計簿管理Webアプリです。  
ユーザーごとに支出データを管理でき、月・カテゴリ別の検索、登録、編集、削除が可能です。

CRUD機能の実装だけでなく、  
**MVC設計・セキュリティ・実務を意識した設計** を重視して開発しました。

---

## 主な機能
- ユーザー登録 / ログイン（Spring Security）
- 支出データの登録・編集・削除
- 月・カテゴリによる検索
- 検索条件を保持した状態での画面遷移
- ユーザーごとのデータ分離（他ユーザーのデータは参照・更新不可）

## 設計・実装のポイント

### MVC設計
- Controller：画面遷移・リクエスト処理を担当
- Service：業務処理の集約
- Repository：DBアクセス（SQL）を集約  
責務が混在しないように役割を分離しています。

---

### セキュリティ対策
- Spring Security を使用したログイン認証
- ログインユーザーIDを元にデータを取得
- **UPDATE / DELETE 時は `WHERE id AND user_id` 条件を必須とし、  
  リクエスト改ざんによる不正更新を防止**

Controller 側の制御だけに依存せず、  
DB更新時にもユーザー制限をかけることで安全性を高めています。

---

### 検索条件保持
一覧画面で指定した「年月・カテゴリ」の検索条件を、  
編集・登録後も維持したまま一覧に戻れるように実装しています。  
ユーザーの操作感を損なわない画面遷移を意識しました。

---

### コード品質への配慮
- 学習メモや不要なコメントは削除し、  
  他者が読むことを前提としたコメントのみを記載
- 不要なロジック・変数の整理
- Repository / Service / Controller で実装方針が混在しないよう統一
